I build practical systems that integrate AI into real workflows: products with users, internal tools with leverage, and control layers that make LLM behavior usable in production.

**Focus**

- AI systems: orchestration, evaluation, structured outputs, and reliability layers around model behavior
- Developer tooling: code understanding, workflow gates, and systems that reduce manual review overhead
- Product systems: marketplaces, intake flows, and multi-user software with real operating constraints

**Selected Work**

- `CrewCast` - Event staffing marketplace built around real-world coordination, supply/demand matching, and multi-user operational flow
- `VibeCheck` - AI code-understanding gate focused on evaluating changes beyond surface-level linting or static checks
- `Neura` - AI-assisted intake system designed around structured collection, workflow routing, and human review loops
- `RUWorkingOut` - Campus product with real usage focus, built around utility, user behavior, and repeat interaction

**What I Like Building**

- Agentic systems that need oversight, routing, and competence boundaries
- Human-in-the-loop workflows where AI assists decisions instead of pretending to replace them
- Products that sit between messy real-world operations and clean user-facing experiences
