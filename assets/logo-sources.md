# Logo Sources

Curated transparent SVG logo sources for the tech shown in the hero.

Notes:
- All links below point to transparent-background SVGs.
- `Wordmark` means the asset includes both the logo and the name.
- `Icon only` means the asset is just the mark. If we use it in the hero, we should pair it with your own text label.
- For protocol/concept items like `OAuth`, `JWT`, `RLS`, `RBAC`, `ETL`, `CI/CD`, there is no single official vendor logo to source without faking it.
